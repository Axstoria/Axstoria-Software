# How to Use the Dragging Movement System

To implement the dragging system located at `Assets/Scripts/Infrastructure/Networking/DraggingMovements.cs`, follow these steps:

## Required Components

Attach the following components to your target GameObject:

1. **Network Identity**
2. **Mesh Collider**
3. **DraggingMovements** (Script)

## Configuration

- Ensure all components are configured with their **default settings**.
- No additional customization is required for basic functionality.
